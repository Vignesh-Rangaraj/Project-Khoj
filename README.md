# 🔍 Project Khoj – SDS Search Tool

A PyQt5-based desktop application to quickly search and access **Safety Data Sheets (SDS)** across multiple languages using a reliable DuckDuckGo API.

---

## 🚀 Overview

Project Khoj is designed to simplify the process of finding Safety Data Sheets (SDS) for chemicals and industrial products. The application provides a fast, user-friendly interface with multilingual search support and secure browsing features.

This tool is useful for:
- Students and researchers  
- Industrial and safety professionals  
- Anyone needing quick access to SDS documents  

---

## ✨ Key Features

- 🌍 Multi-language SDS search  
- ⚡ Fast and reliable results using `duckduckgo_search`  
- 🖥️ Interactive desktop GUI built with PyQt5  
- 🔄 Threaded search for smooth UI experience  
- 🔗 Safe URL validation to block unsafe links  
- 🌐 Open multiple results simultaneously  
- 🌙 Light/Dark mode toggle
